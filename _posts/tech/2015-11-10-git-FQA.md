###`.gitignore`文件对已经纳入git版本管理的文件有效么
* 手动设置不要检查特定文件的更改情况
```
git update-index --assume-unchanged pathname 
```

* 针对个人本地去排除某些文件，不影响.gitignore文件，这个文件一般会git版本管理
```
.git/info/exclude
```
